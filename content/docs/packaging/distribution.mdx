# 分享与分发

## 分享 ZIP 文件

打包好的 `.zip` 文件可以直接发给其他用户。对方收到后：

1. 保存 `.zip` 文件到本地
2. 打开 JSOS
3. 右键 → 安装应用 → 选择文件

## 版本管理

应用版本号写在 `package.json` 的 `version` 字段（或 `jsos.version`）。安装时系统会自动对比版本：

| 情况 | 行为 |
|------|------|
| 全新安装 | 直接安装 |
| 版本更高 | 升级（覆盖安装） |
| 版本更低 | 提示降级 |
| 版本相同 | 覆盖安装 |

## 自动更新

内置应用（系统应用）每次刷新页面时自动检查版本更新。第三方应用暂无自动更新机制，需要用户手动重新安装新版。
