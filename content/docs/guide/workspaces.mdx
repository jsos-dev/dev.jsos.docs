# 工作区与桌面

## 什么是工作区？

工作区就像多个独立的桌面空间。你可以把工作相关的应用放在一个工作区，娱乐相关的放在另一个，互不干扰。

### 操作工作区

- **切换工作区**：点击 Dock 中的工作区图标（💻），或左右拖拽桌面空白处
- **创建工作区**：在工作区切换器中点击「+」
- **重命名工作区**：右键工作区 → 重命名
- **删除工作区**：右键工作区 → 删除（可选择将图标/小组件移动到其他工作区）

> **Home 工作区**是默认工作区，不可删除。

### 跨工作区移动图标

右键应用图标 → **「移动到工作区」** → 选择目标工作区。

### 滑动切换

在桌面空白处左右拖拽，超过一定速度/距离后松手，桌面会平滑切换到相邻工作区。

## 壁纸

每个工作区可以独立设置壁纸。

### 壁纸设置入口

桌面右键 → **「壁纸设置」**

### 壁纸类型

| 类型 | 说明 |
|------|------|
| **默认** | 使用系统默认背景图 |
| **图片** | 本地上传 JPG / PNG / WebP，最大 10MB |
| **视频** | 本地上传 MP4，最大 10MB，无限循环播放 |

### 图片壁纸特效

- **遮罩**：无 / 黑色 / 白色
- **遮罩透明度**：0–100%
- **模糊效果**：0–20px

设置自动保存，切换工作区后恢复对应配置。

## 小组件

小组件是浮动在桌面上的小型面板，显示应用的特定内容。

### 添加小组件

桌面右键 → **「添加小组件」** → 选择需要的小组件。

### 操作小组件

- **移动**：拖拽小组件标题栏
- **调整大小**：拖拽右下角把手
- **右键菜单**：提供更多操作

### 小组件生命周期

- 添加小组件时，后台自动启动所属应用进程
- 关闭应用窗口不会停止小组件
- 移除小组件后，若无其他消费者则自动停止后台进程
