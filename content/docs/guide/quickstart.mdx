# 快速开始

欢迎来到 JSOS！如果你刚刚登录，可能会看到一个干净的桌面——别慌，我们来快速认识一下。

## 桌面布局

```
┌──────────────────────────────────────────────────┐
│                                                    │
│                                                    │
│     📁 文件管理器    ⚙️ 设置                        │
│                                                    │
│     🖥️ 终端         📦 应用管理                     │
│                                                    │
│                                                    │
├──────────────────────────────────────────────────┤
│          🔍  |  💻 工作区  |  ...   Dock           │
└──────────────────────────────────────────────────┘
```

最底部是 **Dock（任务栏）**，类似 macOS 的 Dock 栏。在这里你可以启动应用、切换工作区。

## 打开第一个应用

点击桌面上的**终端**图标，会打开一个终端窗口。这是一个真实的 Linux Shell 环境，你可以在里面执行各种命令：

```bash
echo "Hello JSOS!"
pwd
ls -la
node --version
```

试试在终端里运行 `node -e "console.log('Hi')"`——这就是一个真实的 Node.js 环境。

## 打开文件管理器

点击**文件管理器**图标，你可以浏览系统文件。注意，这里的"文件"是在你的浏览器沙箱中，而不是你电脑上的本地文件。

## 右键菜单

在桌面空白处**右键点击**，会弹出桌面菜单：

- **安装应用** — 从 ZIP 文件安装应用
- **添加小组件** — 在桌面添加小组件
- **壁纸设置** — 更换桌面壁纸
- **全屏/退出全屏** — 使用浏览器的全屏模式

在应用图标上**右键点击**，可以：

- **打开** — 启动应用
- **移动到工作区** — 将图标移到其他工作区
- **属性** — 查看应用信息
- **卸载** — 卸载应用

## 应用窗口操作

每个窗口顶部都有标题栏，除了显示应用名称，还提供以下按钮：

- **关闭** — 关闭应用
- **最小化** — 隐藏窗口（在 Dock 中仍然看得到）
- **最大化** — 窗口最大化
- 拖拽标题栏移动窗口
- 拖拽窗口边缘调整大小
