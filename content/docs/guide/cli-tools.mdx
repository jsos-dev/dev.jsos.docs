# CLI 命令行工具

JSOS v0.6.29+ 提供了命令行工具 `jsos`，让你可以在终端中直接管理系统，无需通过图形界面。

## 工作原理

CLI 工具通过后台运行的 **Daemon 进程**与系统通信：

```
CLI 工具 (jsos)
    ↓ HTTP POST /api/jsos
Daemon (Express, port 1993)
    ↓ stdout/stdin 双向通信
主窗口 (浏览器)
    ↓ 执行 API 调用
    ↓ 返回结果
CLI 工具
```

Daemon 进程在 JSOS 启动时自动运行，提供 HTTP API 端点供 CLI 工具调用。

## 快速开始

在 JSOS 终端中直接使用 `jsos` 命令：

```bash
# 查看系统信息
jsos api getSystemInfo

# 列出已安装应用
jsos app list

# 发送 Toast 通知
jsos toast --title "Hello from CLI" --type success
```

## 全局选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `--port <port>` | Daemon 端口 | `1993` |

## 命令参考

### 通知

| 命令 | 说明 | 示例 |
|------|------|------|
| `toast` | 发送 Toast 通知 | `jsos toast --title "完成" --type success` |

**toast 参数：**

| 参数 | 必填 | 说明 |
|------|------|------|
| `--title <text>` | ✓ | 标题 |
| `--description <text>` | - | 描述 |
| `--type <type>` | - | 类型：`info`、`success`、`warning`、`error` |
| `--duration <ms>` | - | 显示时长（毫秒），默认 3000 |

```bash
jsos toast --title "下载完成" --description "文件已保存" --type success
jsos toast --title "警告" --type warning --duration 5000
```

### 应用管理

| 命令 | 说明 | 示例 |
|------|------|------|
| `app list` | 列出已安装应用 | `jsos app list` |
| `app info <appId>` | 查看应用详情 | `jsos app info dev.jsos.terminal` |
| `app open <appId>` | 打开应用 | `jsos app open dev.jsos.terminal` |
| `app open <appId> --route <route>` | 打开应用并导航 | `jsos app open dev.jsos.settings --route /wallpaper` |
| `app close <appId>` | 关闭应用 | `jsos app close dev.jsos.terminal` |
| `app running <appId>` | 检查应用是否运行 | `jsos app running dev.jsos.terminal` |
| `app install <file>` | 安装应用 | `jsos app install ./my-app.zip` |
| `app uninstall <appId>` | 卸载应用 | `jsos app uninstall my-app` |
| `app uninstall <appId> --delete-data` | 卸载并删除数据 | `jsos app uninstall my-app --delete-data` |

```bash
# 列出所有应用（运行中的应用会标记 ●）
jsos app list

# 安装本地 ZIP 文件
jsos app install ./my-app.zip

# 打开设置应用并导航到壁纸页面
jsos app open dev.jsos.settings --route /wallpaper
```

### 工作区

|------|------|------|
| `workspace list` | 列出工作区 | `jsos workspace list` |
| `workspace current` | 查看当前工作区 | `jsos workspace current` |

| `workspace switch <id>` | 切换工作区 | `jsos workspace switch ws-123` |

jsos workspace list

# 创建新工作区
jsos workspace create "项目开发"
```

jsos wallpaper get

jsos wallpaper reset
```

## 实用示例

### 批量操作

```bash
# 列出所有运行中的应用
jsos app list | grep "●"

# 创建工作区并切换
jsos workspace create "新项目"
jsos workspace switch $(jsos workspace list | grep "新项目" | awk '{print $1}')
```

### 脚本自动化

```bash
#!/bin/bash
# 备份工作区配置
jsos workspace list > workspaces.txt
jsos wallpaper get > wallpaper.json
jsos data export > system-data.json
```

### 系统管理

```bash
# 检查系统状态
jsos api getSystemInfo
jsos app list
jsos workspace list

# 重置系统
jsos wallpaper reset
jsos theme set system
jsos locale set zh-CN
```

## 故障排除

### 无法连接 Daemon

如果遇到 `ECONNREFUSED` 错误，说明 Daemon 进程未运行：

```bash
# 检查 Daemon 状态
jsos api getSystemInfo

# 如果 Daemon 未运行，重启 JSOS 系统
# 在图形界面中刷新页面即可
```

### 命令无响应

如果命令长时间无响应，可能是 Daemon 处理超时：

```bash
# 使用默认超时（30 秒）
jsos api getApps

# 如果数据量大，可能需要等待
jsos data export
```
