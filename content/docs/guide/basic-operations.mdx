# 基本操作

## 窗口管理

### 打开窗口

点击桌面图标启动应用。GUI 应用会打开一个带标题栏的窗口，CLI 应用打开一个终端窗口。

### 移动窗口

拖拽标题栏任意位置，窗口跟随鼠标移动。

### 调整大小

拖拽窗口的**四边或四角**调整大小。

### 窗口按钮

每个窗口右上角有三个按钮：

- **×** 关闭 — 终止应用进程
- **—** 最小化 — 窗口隐藏到后台，Dock 中仍可见
- **□** 最大化 — 铺满屏幕（再次点击恢复大小）

### CLI 应用的终端窗口

CLI 应用没有图形界面，整个内容区域就是终端。窗口底部有一条状态栏：

- 进程**运行中**时显示进程信息
- 进程**退出**后显示退出码，提供重启 / 关闭按钮
- 如果退出码非零，终端输出会添加红色提示

## Dock（任务栏）

底部 Dock 栏提供以下功能：

| 元素 | 说明 |
|------|------|
| **搜索图标** (`🔍`) | 打开应用启动器，搜索和启动已安装应用 |
| **工作区图标** (`💻`) | 打开工作区切换器，创建/切换工作区 |
| **运行中的应用** | 显示已启动的应用窗口，点击切换/激活 |
| **设置** | 点击 Dock 中的设置图标打开系统设置 |

## 桌面右键菜单

在桌面空白处右键点击：

- **安装应用** — 从 ZIP 文件安装应用（详见[安装应用](/docs/guide/installing-apps)）
- **添加小组件** — 在桌面添加小组件
- **壁纸设置** — 更换壁纸（图片/视频/默认，支持遮罩和模糊）
- **全屏** / **退出全屏** — 使用浏览器 Fullscreen API

## 图标右键菜单

在桌面应用图标上右键点击：

- **打开** — 启动该应用
- **移动到工作区** → 子菜单列出所有工作区，点击即可移动
- **属性** — 查看应用详情（ID、版本、类型、端口等）
- **卸载** — 将该应用从系统中移除（系统应用不可卸载）

## 桌面拖拽操作

在桌面空白处**左右拖拽**，可以在工作区之间切换。拖拽到屏幕边缘（距离边缘 80px 以内）时，松手后会自动切换到相邻工作区。
